⚔️ THE GAUNTLET ⚔️

Epic Multi-Stage Adventure
🏰 Castle Gates → 🗻 Piedmont Hills → 🌲 Forest Depths → ⛰️ Mountain Pass → 🏜️ Desert Oasis

🎮 Combat Controls

WASD / Arrow Keys - Move hero
SPACE - Jump (distance: 4, height: 6)
E - Strike attack (range: 2)
R - Reset current stage

🏰 Stage Navigation

N - Next stage (manual advance)
P - Previous stage
Auto - Stages advance when cleared
Timeout - Auto-advance after 150 ticks

🎯 Gauntlet Features

  • Dynamic Battlefields: Each stage has unique dimensions and layouts
  • Background Stitching: Seamless environment transitions
  • Progressive Difficulty: Enemy types and counts scale with progression
  • Stage-Specific Enemies: Castle soldiers → Forest mages → Desert nomads
  • Epic Scale: 640x400 equivalent with double-wide battle strips
  • Dramatic Effects: Stage transitions, victory celebrations, completion fireworks
